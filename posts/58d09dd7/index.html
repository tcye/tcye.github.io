软件配置

  1. CLion, PyCharm:

    • 下载后分别解压到/opt/clion /opt/pycharm

    • Help->Edit Custom Options，加入

    • Tools->Create Desktop Entry

    • licence: 账号密码见网易云笔记

  2. QtCreator:

    • 编辑 /usr/share/applications/DigiaQt-qtcreator-community.desktop

  3. VMWare Player

  4. l2tp vpn

  5. vim && font && ssh

    • 参考 https://github.com/tcye/editor_config
